# Trafficker

[![Python 3.6](https://img.shields.io/badge/Python-2.7-bule.svg)](http://www.python.org/download/)

任意流量发包，pcap解析工具

## 安装


```shell
git clone https://github.com/LyleMi/Trafficker.git
python setup.py install
```

## 特性

### 支持协议

* arp
* dns
* icmp
* ip
* pop
* smtp
* tcp
* udp
* vlan
