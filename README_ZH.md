# Trafficker

[![Python 2.7](https://img.shields.io/badge/Python-2.7-bule.svg)](http://www.python.org/download/)

任意流量发包，pcap解析工具

## 安装


```shell
pip install -r requirements.txt
git clone https://github.com/LyleMi/Trafficker.git
sudo python app.py
```

## 特性

### 支持协议

* ip
* arp
* icmp
* tcp
* udp
* http

## 其他功能

* pcap 包解析
